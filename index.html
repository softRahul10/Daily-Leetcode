Q1. Swap Nodes In Linked List

Solution

                                Node swapPairs(Node head) {
                                    Node dummy = new Node(0);
                                    Node curr = head;
                                    Node prev = dummy;

                                    while(curr!=null && curr.next != null) {
                                        Node next;
                                        prev.next = curr.next;
                                        next = curr.next.next;
                                        curr.next.next = curr;
                                        prev = curr;
                                        curr = next;
                                    }
                                    prev.next = curr;
                                    return dummy.next;
                                }
